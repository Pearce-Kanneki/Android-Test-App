# Android-Test-App.
### 此專案用於練習Android App測試用途
[自動化工具Wiki 說明](https://hackmd.io/@9jq2Y4QbRY65xEzdsgCy6A/r1ew6F9Hq)

練習項目
1. Unit Test
2. UI Test
3. e2e Test (End To End Test)
4. 自動化測試(Appium)
5. BDD
6. DevOps (Jenkins)


App功能說明

List頁: List表(初始化10筆)，與有ADD Button  
Account頁: 登入驗證檢查
